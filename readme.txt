Ever wanted a fresh start on your youtube channel? Its impossible when you'd have to manually unsubscribe 
from channels and unlike watched videos.

This tool will automate the whole process for you!

Usage:
1) Follow this tutorial to get client secret credentials: https://developers.google.com/youtube/registering_an_application
2) Set the path in the python script
3) Run python3 main.py
4) Step away and come back in a few minutes! The process takes a while due to the high volume of API requests
 but can be ran in the background

Implementation Details:
VIA Youtube Data API
